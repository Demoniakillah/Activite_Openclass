Liste des commande de GIT

  • git init => Initialiser le répertoire courant comme repository git
  • git status => Status actuel des commandes
  • touch nom_du_fichier => Nouveau fichier
  • git add nom_du_fichier => Ajouter à l'index de git
  • git commit -m <"actions réalisée"> => Validation du fichier
  • git commit -am <"actions réalisée"> => Ajout et Validation en une seule commande ( uniquement après premier ajout à l'index)
  • git log => Affichage de toutes le modifications triées par date (du plus récent au plus ancien)
  • git checkout <"code sha1"> => Se positionner sur un commit donné
  • git checkout master => Revenir à la dernière mofication du projet
  • git clone => Récupérer TOUT le repository
  • git push master => Envoyer son code
  • git pull master => Récupérer uniquement les modifications
